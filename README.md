
# Dogipedia    
![com.anya.dogipedia.screenshot.icon](https://raw.githubusercontent.com/AnnaBron/Dogipedia/master/app/src/main/res/mipmap-xxxhdpi/ic_launcher.png)  

Sample Android app in [Kotlin](https://kotlinlang.org/) for viewing many dogs breeds and images based on [Dog API](https://dog.ceo/dog-api/).  
  
## Building the app  
* Gradle build tools  
* Requires [Java 8](https://java.com/en/download/faq/java8.xml)  
* Android studio v4+
  

## Architecture  
* MVVM (Model-View-ViewModel)  
* [Android Arhitecture Components](https://developer.android.com/topic/libraries/architecture/)  
* [LiveData](https://developer.android.com/reference/android/arch/lifecycle/LiveData)  
* [CoRoutines](https://developer.android.com/kotlin/coroutines?gclid=Cj0KCQiA1pyCBhCtARIsAHaY_5efyKsahOpTlWotbodUwJlRrPUZY7-WFq78YAB8f6pqb6Rqswxu-DUaArDLEALw_wcB&gclsrc=aw.ds)  
* [Data Binding](https://developer.android.com/topic/libraries/data-binding/)  
* [Android Jetpack/Navigation](https://developer.android.com/topic/libraries/architecture/navigation/) - Android navigation framework  
  
## Dependency management  
* [Google/Dagger/Hilt](https://dagger.dev/hilt/) - Dependency injection  
  
## Network  
* [Square/Retrofit](https://github.com/square/retrofit) - HTTP RESTful connections  
* [OkHttp 3](https://square.github.io/okhttp/3.x/okhttp/)  
* [Square/Moshi](https://github.com/square/moshi) - JSON adapter
* [Merlin](https://github.com/novoda/merlin) - Observes network connection status & gives callbacks
